# Codveda Data Cleaning & EDA Project

## Overview
This project was completed as part of the Codveda Technologies Data Analytics Internship.

The project focuses on:
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Correlation Analysis

---

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- GitHub

---

## Tasks Completed

### Task 1: Data Cleaning and Preprocessing
- Loaded dataset using pandas
- Fixed delimiter and formatting issues
- Assigned proper column names
- Checked missing values
- Checked duplicate rows
- Standardized column names
- Exported cleaned dataset

### Task 2: Exploratory Data Analysis (EDA)
- Generated summary statistics
- Created histograms
- Created boxplots
- Created scatterplots
- Built correlation heatmap
- Extracted insights from data

---

## Key Insights
- Houses with more rooms tend to have higher prices.
- LSTAT negatively affects house prices.
- Strong correlations exist between several features.
- Outliers are visible in some numerical columns.
