# Zecpath AI System

## Project Overview
This repository contains the AI system for Zecpath, designed to automate and optimize the hiring lifecycle using modular AI components.

## Folder Structure
zecpath-ai-system/
│
├── ats_engine/ # ATS scoring logic
├── screening_ai/ # Screening AI modules
├── interview_ai/ # Interview intelligence modules
├── scoring/ # Final decision and scoring logic
├── parsers/ # Resume parsing logic
├── utils/ # Utilities (logging, helpers)
├── tests/ # Unit tests
├── data/ # Sample and processed data
├── logs/ # System logs
├── venv/ # Virtual environment
├── main.py # System entry point
├── requirements.txt # Dependencies


## How to Run
1. Activate virtual environment:
venv\Scripts\activate


2. Run the system:
python main.py


## Logging
All AI activities are logged to:
logs/ai_system.log


## Testing
Sample test available:
tests/test_ats.py


## Code Standards
- Python PEP8 style followed
- Modular design per AI component
- Clear function naming
- Logging enabled for major actions
- Unit tests for core logic
