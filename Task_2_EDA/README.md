# Exploratory Data Analysis (EDA) - House Price Dataset

## Objective
Perform exploratory data analysis on the House Price dataset to identify patterns, trends, correlations, and summary statistics.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
1. Loaded dataset using pandas
2. Checked dataset structure and statistics
3. Visualized distributions using histograms
4. Created boxplots to detect outliers
5. Created scatterplots to identify relationships
6. Generated correlation heatmap
7. Extracted insights from the dataset

## Key Insights
- Houses with more rooms tend to have higher prices.
- LSTAT negatively impacts house prices.
- Several features show strong correlations.
- Outliers exist in some numerical columns.
